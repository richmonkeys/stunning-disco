## Countries API

API to retrive a list of countries or a single country.

`Country` typescript definition:

```ts
type Country = {
  countryName: string
  countryCode: string
  countryNameTranslations: {
    [languageCode: string]: string
  }
  nativeCountryName: string
  continentCode: string
  continentName: string
  capital: string
  capital: string
  isoAlpha3: string
  isoNumeric: string
  areaInSqKm: number
  fipsCode: string
  languageCodes: string[]
  currencyCode: string
  currencyName: string
  currencySymbol: string
  postalCodeFormat: string
  e.164: string
  timezones: {
    timezone: string
    name: string
    abbreviation: string
    gmtOffset: number
    gmtOffsetName: string
  }[]
  latitude: string
  longitude: string
  population: number
  north: number
  east: number
  south: number
  west: number
  emoji: string
  emojiUnicode: string
}
```

## States API

API to retrive a list of states or a single state.

`State` typescript definition:

```ts
type State = {
  stateName: string
  stateCode: string
  countryCode: string
  latitude: string
  longitude: string
}
```

---

### Get list of states by countryCode

**GET** `/api/states?countryCode={countryCode}`

**Query**

```ts
countryCode: string // required
```

**Example Returns**
```json
[
  ...
  {
    "stateName": "Kuala Lumpur",
    "stateCode": "MY-14",
    "countryCode": "MY",
    "latitude": "3.13900300",
    "longitude": "101.68685500"
  },
  ...
]
```

---

### Get a state by stateCode

**GET** `/api/states/:stateCode`

**Query**

```ts
stateCode: string // required
```

**Example Returns**
```json
{
  "stateName": "Kuala Lumpur",
  "stateCode": "MY-14",
  "countryCode": "MY",
  "latitude": "3.13900300",
  "longitude": "101.68685500"
}
```

## Cities API

API to retrive a list of cities.

*Note that getting a single city via this API is not possible.*

`State` typescript definition:

```ts
type City = {
  cityName: string
  countryCode: string
  stateCode: string
  latitude: string
  longitude: string
}
```

---

### Get list of cities by either countryCode or stateCode

Only accepts either countryCode or stateCode as query, but not both.

**GET** `/api/cities?countryCode={countryCode}`

**Query**

```ts
// only one of the following
countryCode: string
stateCode: string
```

**Example Returns**
```json
[
  ...
  {
    "cityName": "Kuala Lumpur",
    "countryCode": "MY",
    "stateCode": "MY-14",
    "latitude": "3.14120000",
    "longitude": "101.68653000"
  },
  ...
]
```
